# Onyekwere Akym Uche | The OAU Foundation

Dedicated to impacting humanity through humanitarian efforts, sustainable development, and empowerment programs.

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```

### Development

Start the development server:
```sh
npm run dev
```

### Build

Build for production:
```sh
npm run build
```

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
